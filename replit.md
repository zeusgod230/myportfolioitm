# Simple Portfolio Website

## Overview
A clean and professional portfolio website built with vanilla HTML, CSS, and minimal JavaScript. The site features three main pages: Home, About, and Contact. Designed to be simple, easy to understand, and ready for deployment on Netlify.

## Project Structure
```
.
├── index.html      # Home page with hero section and intro cards
├── about.html      # About page with personal story and skills
├── contact.html    # Contact page with form and contact details
├── style.css       # Main stylesheet with responsive design
├── script.js       # JavaScript for mobile menu and form handling
└── replit.md       # Project documentation
```

## Features
- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Clean Navigation**: Sticky navbar with mobile hamburger menu
- **Home Page**: Hero section with call-to-action buttons and service cards
- **About Page**: Personal story section with skills showcase
- **Contact Page**: Contact form with validation and social media links
- **Modern UI**: Gradient backgrounds, smooth transitions, and hover effects

## Technologies Used
- HTML5 for structure
- CSS3 for styling (no frameworks)
- Vanilla JavaScript for interactivity (no libraries)

## Customization Guide
To personalize this portfolio:

1. **Update Personal Information**:
   - Replace "Your Name" in index.html
   - Add your email, phone, and location in contact.html
   - Update social media links in contact.html

2. **Add Your Photo**:
   - Replace the placeholder in about.html with an actual image
   - Update the `.image-placeholder` section in about.html

3. **Customize Colors**:
   - Main gradient: Change in `.hero` and `.page-header` (currently purple)
   - Primary color: Update `.btn-primary` and links (currently blue)
   - Edit the `style.css` file to match your brand

4. **Update Content**:
   - Modify the intro text on the home page
   - Update your story and skills in about.html
   - Change the service cards to match what you offer
